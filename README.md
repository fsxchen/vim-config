# VIM 7.4 python IDE
## 安裝

```
git clone https://github.com/fsxchen/vim-config.git ~/.vim
```



```
sudo apt-get install ctags
cd ~/.vim
sh install.sh
```

## 安裝vim插件

進入`vim`命令行模式

```
:BundleInstall
```

+ F2: 打开/关闭目录树
+ F3: 自动插入Title
+ F5: 运行
